# PRPs — Product Requirements Prompts

Feature planning for Chasqui lives here. We use the **prp-manager** skill.

- Install (if needed): `npx skills add https://github.com/willywg/prp-manager --skill prp-manager`
- Write/refine a PRP **before** implementing a non-trivial feature, then execute it in the relevant service repo (`core`, `admin`, `whatsapp`).

```
PRPs/
├── templates/   # PRP templates (from prp-manager)
└── *.md         # one PRP per feature
```
